## 系列概览

这个系列从零开始手把手教你怎么写一个 Agent 系统：

1. **范式革命**：从"人驱动代码"到"模型驱动代码"的思维转变
2. **工具调用**：让 LLM 能调用外部函数（Function Calling）
3. **Agent Loop**：感知→决策→行动→反思的循环
4. **实战**：用两步做一个财报研报 Agent
5. **设计范式**：Agent 的架构模式选择
6. **记忆系统**：短期记忆、长期记忆、摘要记忆
7. **上下文工程**：Context Window 管理、压缩、裁剪
8. **外接知识库**：RAG 集成
9. **浏览器自动化**：让 Agent 操作网页
10. **Skills 系统**：可复用的 Agent 技能
11. **生产级系统**：部署、监控、安全、评估
