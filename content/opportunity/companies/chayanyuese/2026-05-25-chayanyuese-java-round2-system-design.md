## 核心场景题（饮品订单系统）

### 8. 分库分表后订单查询优化
按用户 ID 分库后，如何高效实现"查询某门店所有订单"？

### 9. 服务雪崩防护
促销活动导致订单服务调用积分服务超时，如何设计熔断降级策略？

### 10. 分布式 ID 生成器选型
订单 ID 需要包含门店编号和时间戳，如何设计？

### 11. RPC 调用超时治理
积分服务响应慢导致订单服务线程池耗尽，如何优化？

### 12. MySQL 索引失效案例
```sql
SELECT * FROM orders WHERE store_id = 1 AND status = 'paid' ORDER BY create_time DESC;
```
如何优化？

### 13. Redis 缓存穿透防御
如何防止恶意查询"已下架饮品"导致 DB 压力？

### 14. 饮品库存扣减方案
先查缓存再扣减 DB，如何解决并发下的数据不一致？

### 15. ES 搜索优化
如何实现"根据用户位置推荐最近门店"的毫秒级响应？

### 16. 多级缓存架构
本地缓存（Caffeine）与 Redis 如何协同保证菜单数据一致性？

### 17. 订单状态异步通知
用户支付成功后，如何保证短信、APP 推送、微信模板消息的最终一致性？

### 18. Kafka 消息积压处理
门店打印机离线导致订单打印消息堆积，如何恢复后快速消费？

### 19. 延迟队列实现
如何实现"订单 15 分钟未支付自动取消"？

### 20. 消息幂等设计
积分系统重复收到订单完成消息，如何避免用户积分加倍？

### 21. 批量操作优化
500 家门店的日结算报表如何通过消息队列异步生成？

### 22. ThreadLocal 内存泄漏排查
订单链路追踪中使用的 ThreadLocal 为何导致 Full GC？

### 23. ConcurrentHashMap 实战
如何设计一个线程安全的"门店实时订单计数器"？

### 24. JVM 调优案例
线上订单服务频繁 Full GC，如何通过 GC 日志定位问题？（G1 vs ZGC 场景选择）

### 25. CompletableFuture 应用
如何并行查询"用户信息、优惠券、库存"后组装订单确认页？

### 26. 锁粒度优化
同步代码块 `synchronized(storeId)` 在高并发场景下导致性能瓶颈，如何改进？

### 27. CPU 100% 排查
订单服务某台机器 CPU 持续 100%，如何快速定位？（arthas thread -n 3）

### 28. 接口性能优化
`/api/menu/list` 响应慢，从 SQL、缓存、网络多维度给出优化方案

### 29. OOM 问题复现
如何模拟一个"订单对象堆积导致堆内存溢出"的场景？

### 30. 数据库连接池配置
Druid 连接池在高峰期出现 getConnection 超时，如何调整参数？

### 31. 日志治理方案
订单系统每日产生 100GB 日志，如何设计采集、存储和检索架构？
