# 字节 AI Agent 后端一面面经（多人汇总）

> **来源**：小红书 xhslink.com/o/3FbW2c8Vezp
> **抓取时间**：2026-06-07
> **特点**：分多位求职者汇总，技术时长约 40～120min，覆盖面杂

---

## 一面（40～120min）

### 项目深挖
1. 自我介绍，围绕实习、LLM（大语言模型）/RAG（检索增强生成）/Agent 项目展开
2. 梳理项目难点，追问落地里最棘手的环节
3. 讲解 Transformer（变换器）完整原理与执行流程

### 大模型应用
4. MCP（模型上下文协议）、RAG、Skill 工具三者概念与落地区别
5. 项目选用 LangGraph 而非原生 Agent 框架的理由
6. 超长上下文冗余怎么优化处理
7. ReAct（推理+行动）机制原理与落地细节

### Java 后端八股
8. HashMap 底层结构、哈希冲突与扩容逻辑
9. JVM 各类 GC 算法、主流垃圾回收器适配场景
10. Spring AOP 底层原理，JDK 动态代理 & CGLIB 代理差异、AOP 失效特殊场景
11. MySQL 四大事务隔离级别，MVCC（多版本并发控制）怎么实现各级隔离效果
12. Redis 常用数据类型 + 业务场景，Redis 分布式锁实现思路

### 系统设计
13. 数据库表结构整体设计思路
14. 购物车架构设计，未登录用户跨设备同步购物车数据方案
15. 优惠券、满减活动的规则设计与库存管控

### 计网 & 操作系统
16. OS 死锁定义与产生条件
17. URL 输入到页面展示全链路（DNS 解析、TCP、HTTP 整套流程）

### 手撕算法（不同人抽到不同题目）
18. 二叉树路径和等于 K，输出全部合法路径
19. 最长回文子串（中心扩散，补充 DP 写法）
20. 数组找局部峰值元素
21. O(1) 原地：奇数放奇数下标、偶数放偶数下标
22. 解码数字串 LC91 Decode Ways

---

## 二面 & 三面汇总

- 二面大多先双向自我介绍，部分场次仅现场手写算法，沟通简短四十分钟结束
- 三面以 BQ（行为面试）+ 智力题穿插提问，全程无代码手撕
- 附加闲聊：有人原本意向算法岗，面试后被内部调剂至别的业务线；部分面试偏向 KPI 招聘

---

## 覆盖知识点

| 方向 | 考点 |
|------|------|
| **AI Agent** | MCP vs RAG vs Skill、LangGraph 选型、ReAct 机制、超长上下文优化 |
| **Java** | HashMap、JVM GC、Spring AOP、MySQL 隔离级别+MVCC、Redis 分布式锁 |
| **系统设计** | 购物车架构、优惠券规则、库存管控 |
| **网络** | URL→页面全链路、死锁 |
| **算法** | 二叉树路径、回文子串、峰值元素、奇偶重排、Decode Ways |

---

## 面试特点

**覆盖面极杂**：Agent/RAG 落地细节 + Java 后端八股 + 系统设计 + 手撕算法，四条线并行考察。既要吃透大模型应用，后端基础也不能薄弱。
