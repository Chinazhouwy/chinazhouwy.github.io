上一篇我们介绍了如何调用模型 API，这一篇我们来学习如何让模型调用工具。

单纯只是调用模型 API 的 Agent 只是 ChatBot。

会调用工具的 Agent，才是真正的 Agent。

先看代码

我给你看一段最简单的代码，还是几十行。

总结

现在的模型已经支持了原生调用工具的能力，我们只需要在调用模型时，传入 tools 参数，就可以了。

从聊天到调用工具，已经有了 Agent 的雏形，但是现在只能完成单步任务。
