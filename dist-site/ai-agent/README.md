## 高频考点Top10

1. **Agent架构分层**（接入层→编排层→能力层→基础设施层）
2. **Workflow vs ReAct vs 自主决策**的取舍
3. **Memory三层设计**（短期/摘要/长期）及触发机制
4. **RAG全链路**（解析→切分→混合检索→Rerank）
5. **Multi-Agent编排**（中心化 vs 去中心化）
6. **Prompt工程化**（版本管理、A/B测试、自动评估）
7. **多轮对话管理**（指代消解、状态对齐、历史压缩）
8. **Agent观测体系**（链路追踪、Token监控、业务指标）
9. **延迟vs效果取舍**（分级响应、小模型分流、缓存）
10. **评测与迭代**（评测集构建、Badcase回流、SFT vs Reward Model）
