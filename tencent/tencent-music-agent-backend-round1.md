# 腾讯音乐后端（Agent）开发一面面经

> 来源：小红书 | 日期：2026-04-28 | 提取日期：2026-05-22
> 岗位写的是后端开发，实则问的都是 Agent 相关内容。面试仅半小时，节奏很快，问完一个立马接下一个。

## 项目一：AI Agent 项目

1. **自我介绍**

2. **项目是前后端都有吗？介绍一下完整的运行流程**

3. **项目是你独立开发的吗？你主要负责哪一部分？**

4. **开发过程中有没有用到 AI 辅助？**

5. **介绍一下记忆模块怎么做的。**

6. **那会话记忆超出上下文窗口怎么办？**
   - 参考：滑动窗口、总结压缩、持久化存储

7. **用的什么向量数据库？还了解哪些向量数据库？**

8. **后续项目如果要改进，你有哪些改进方向？**

9. **总结压缩历史对话成摘要的 prompt 你是怎么设计的？**

10. **用的什么 AI 框架开发？一共设计了几个 agent？用了几种大模型？**

11. **意图识别模块怎么做的？**

12. **质量检查怎么做的？**

13. **如果要把项目改造成能完整生成前后端的平台，你会怎么做？**

14. **部署环节你是怎么做的？**

## 考察重点

- Agent 核心模块：记忆、意图识别、质量检查
- 上下文窗口管理策略
- Prompt 设计能力
- 向量数据库理解
- 项目架构和后续演进思路
- 部署能力
