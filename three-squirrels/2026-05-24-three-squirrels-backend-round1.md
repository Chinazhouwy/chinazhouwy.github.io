## 场景题

### 1️⃣ 618 大礼包怎么防超卖？

电商核心痛点，高频面试题。

**回答要点：**
- **Redis 预扣 + Lua 原子性**：用 Lua 脚本执行 DECR 库存操作，保证原子性
- **异步落库 + 消息队列削峰**：扣减成功后发 MQ 异步落库，避免数据库写压力
- **数据库乐观锁兜底**：最终一致性保证，防止 Redis 宕机导致数据不一致

### 2️⃣ 礼盒年货节销量下滑，有限预算怎么挽回？

业务场景题，考察数据分析 + 运营策略能力。

**回答要点：**
- **数据分析**：拉取流量来源 + 用户画像，找流失原因
- **定向投放**：针对高潜力人群做精准投放
- **优化选品组合**：根据历史销售数据调整礼盒搭配
- **社群私域裂变**：低成本获客，老客带新客

### 3️⃣ 如何提升抖音直播间成交？

直播电商场景，考察对直播生态的理解。

**回答要点：**
- **直播前预热种草**：短视频 + 预告引流
- **中控节奏引导**：话术、秒杀节奏、互动引导
- **选品组合策略**：引流品 + 爆款钩子搭配

## 八股题

| 问题 | 回答要点 |
|---|---|
| Redis 一般什么时候用？ | 缓存热点数据、分布式锁扣库存、会话管理 |
| MySQL 单表太大怎么办？ | 分库分表（按用户 id 哈希拆分），选好分区键避开热点倾斜 |
| 负载均衡器挂了怎么办？ | 业务冗余接入，多节点部署 + 健康检查自动踢除故障节点 |
| 权限系统数据库表怎么设计？ | **RBAC + ABAC**：用户表、角色表、权限表、用户角色关联表 |
| 物流系统架构了解吗？ | 供应链核心，用户需求导向，多看物流方向专业知识 |

## 总结与建议

三只松鼠虽然是零食电商，技术面试**一点都不水**。全是高并发 + 电商场景的实战题：

- **库存扣减**（超卖/防超卖）
- **流量优化**（销量下滑怎么救）
- **直播转化**（直播间成交提升）

这些都是电商核心痛点，面试官关注你是否**真正处理过线上问题**，不是背八股就能过的。建议准备电商方向的同学重点练这三个方向。
