中文隐私政策

本政策适用于 Microsoft Edge、Google Chrome 及其他兼容浏览器中的 “Markdown 阅读器”扩展。使用本扩展即表示你了解本政策中说明的数据处理方式。

1. 扩展的单一用途

本扩展用于读取、渲染、编辑、搜索、打印和导出由用户主动打开或选择的本地 Markdown、Markdown Text 及文本文件。扩展不会在此用途之外处理文档内容。

2. 扩展会访问的数据

本地文件内容

当用户打开本地文件 URL 或在查看器中选择文件夹时，扩展会访问相应文件的文本、图片引用和链接，以便在浏览器中显示和编辑文档。这些内容仅在用户设备上处理，不会发送给开发者或第三方。

浏览器本地存储

扩展可能使用浏览器提供的 storage 权限保存主题、语言、面板宽度、编辑器设置、最近文件或文件夹提示，以及更新提醒状态。这些设置仅保存在用户的浏览器中。

3. 数据收集、传输与共享

  • 不收集或上传本地文档内容。
  • 不出售、出租或向第三方共享用户数据。
  • 不使用分析服务、广告 SDK 或第三方跟踪器。
  • 不使用远程托管的代码执行扩展功能。
  • 不将用户数据用于信用评估、借贷或与扩展单一用途无关的目的。

4. 更新检查

Microsoft Edge、Google Chrome 或其他浏览器可能通过其官方扩展更新服务检查和安装新版本。更新检查不会由本扩展附带或上传用户的文档内容。

5. 数据保留与删除

本扩展不在开发者服务器上保留用户数据。用户可以通过清除扩展存储或卸载扩展删除浏览器中保存的设置。用户的原始本地文件始终由用户自行管理。

6. 政策变更与联系

如本扩展的数据处理方式发生变化，本页面将同步更新。有关隐私或扩展支持的问题，请通过 zhouwy.top 联系开发者。

Privacy Policy in English

This policy applies to the Markdown Viewer extension for Microsoft Edge, Google Chrome, and other compatible browsers.

1. Single Purpose

The extension reads, renders, edits, searches, prints, and exports local Markdown and text files that the user explicitly opens or selects. It does not process document content for any unrelated purpose.

2. Data the Extension Accesses

The extension accesses the text, image references, and links in local files selected by the user so it can display and edit those documents. All document processing occurs locally on the user's device. Document content is never sent to the developer or a third party.

Browser storage may contain local preferences such as theme, language, panel widths, editor settings, recent file or folder hints, and update reminder state. These settings remain in the user's browser.

3. Collection, Transfer, and Sharing

  • The extension does not collect or upload local document content.
  • User data is not sold, rented, or shared with third parties.
  • No analytics, advertising SDKs, or third-party trackers are used.
  • No remotely hosted code is used to provide extension functionality.
  • User data is not used for credit, lending, or unrelated purposes.

4. Updates, Retention, and Deletion

The browser may use its official extension update service to check for and install new versions. The extension does not attach document content to update checks. The developer does not retain user data on a server. Users can remove locally stored settings by clearing extension storage or uninstalling the extension.

5. Changes and Contact

This page will be updated if the extension's data practices change. For privacy questions or extension support, contact the developer through zhouwy.top.
