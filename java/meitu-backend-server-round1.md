# 美图后端服务器开发一面面经

> 来源：小红书 | 日期：2026-05-26 | 状态：✅ 已过

## 面试体验

面试官人很好，全程微笑，卡壳了也说不用紧张。虽然简历上没有 AI 项目也被问了 AI 问题，没答完整以为挂了，结果过了。

## 面试题

1. 自我介绍
2. 八股拷打（比较简单）
3. HashMap 底层原理以及 put 流程？
4. 怎么解决 HashMap 线程不安全问题？
5. JMM（Java 内存模型）？
6. 联合索引？以及索引失效的原因？
7. Spring 循环依赖？
8. 三级缓存的底层原理？
9. Redis 三大问题以及解决方案？（穿透/击穿/雪崩）
10. 消息队列消息消费失败怎么处理？
11. 项目拷打
12. AI 相关

## 面试总结

- 一面难度总体偏低，以八股为主
- HashMap + JMM + 索引 + Spring循环依赖 + Redis 是核心考察点
- 即使没有 AI 项目经验也被问到 AI 问题，建议提前准备基础认知
